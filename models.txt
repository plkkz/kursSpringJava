Ученики (Students)

student_id (PK) — уникальный идентификатор ученика
first_name — имя ученика
last_name — фамилия ученика
birth_date — дата рождения
class_id (FK) — идентификатор класса, в котором учится ученик
Классы (Classes)

class_id (PK) — уникальный идентификатор класса
class_name — название класса (например, "10А")
teacher_id (FK) — идентификатор учителя, который ведет класс
Учителя (Teachers)

teacher_id (PK) — уникальный идентификатор учителя
first_name — имя учителя
last_name — фамилия учителя
subject — предмет, который преподает учитель
Предметы (Subjects)

subject_id (PK) — уникальный идентификатор предмета
subject_name — название предмета
Оценки (Grades)

grade_id (PK) — уникальный идентификатор оценки
student_id (FK) — идентификатор ученика, которому поставлена оценка
subject_id (FK) — идентификатор предмета
grade — оценка (например, 5, 4, 3 и т.д.)
date — дата выставления оценки
Посещения (Attendance)

attendance_id (PK) — уникальный идентификатор посещения
student_id (FK) — идентификатор ученика
date — дата посещения
status — статус (например, "Присутствует", "Отсутствует", "Опоздал")
Домашние задания (Homework)

homework_id (PK) — уникальный идентификатор домашнего задания
subject_id (FK) — идентификатор предмета
title — название домашнего задания
description — описание задания
due_date — дата сдачи задания
Оценки за домашние задания (Homework_Grades)

homework_grade_id (PK) — уникальный идентификатор оценки за домашнее задание
homework_id (FK) — идентификатор домашнего задания
student_id (FK) — идентификатор ученика
grade — оценка за домашнее задание
2. Связи между сущностями
Ученики и Классы:

Один класс может содержать много учеников (1:N).
Один ученик принадлежит только одному классу (N:1).
Классы и Учителя:

Один учитель может вести много классов (1:N).
Один класс ведет только один учитель (N:1).
Ученики и Оценки:

Один ученик может иметь много оценок (1:N).
Оценка относится только к одному ученику (N:1).
Предметы и Оценки:

Один предмет может иметь много оценок (1:N).
Оценка относится только к одному предмету (N:1).
Ученики и Посещения:

Один ученик может иметь много записей о посещаемости (1:N).
Каждая запись о посещаемости относится только к одному ученику (N:1).
Предметы и Домашние задания:

Один предмет может иметь много домашних заданий (1:N).
Каждое домашнее задание относится только к одному предмету (N:1).
Ученики и Оценки за домашние задания:

Один ученик может иметь много оценок за домашние задания (1:N).
Каждая оценка за домашнее задание относится только к одному ученику (N:1).
Домашние задания и Оценки за домашние задания:

Одно домашнее задание может иметь много оценок (1:N).
Каждая оценка за домашнее задание относится только к одному домашнему заданию (N:1).
